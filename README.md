# Portfólio de Identidade Visual ✨

Este é o repositório do meu portfólio digital, focado em exibir projetos de branding, identidade visual e direção criativa para marcas que desejam se destacar.

## 🚀 Tecnologias Utilizadas

- **HTML5:** Semântica estrutural
- **CSS3:** Estilização responsiva (Mobile First), CSS Variables e animações fluidas
- **JavaScript (ES6):** Manipulação de DOM e lógica matemática do Canvas

## 💻 Como visualizar

O projeto está hospedado no GitHub Pages e você pode acessar o portfólio ao vivo clicando no link disponível na página inicial do repositório (ao lado, em "Environments" ou "Deployments").
